
# Movie Buzz-Ticket Booking Website (MERN Stack)

A full-stack Ticket Booking Website built using the MERN stack (MongoDB, Express.js, React, Node.js).
This project allows users to browse events, book tickets, manage reservations, and handle secure authentication.


## Features

- 🔑 User Authentication & Authorization (JWT / bcrypt)
- 🎫 Event Listings & Ticket Booking
- 📅 Booking History & Management
- 💳 Payment Integration (Stripe) – in progress
- 👨‍💻 Admin Panel for event & booking management
- 📱 Responsive UI with React + Tailwind CSS




## Tech Stack

**Client/Frontend:** React, Redux, TailwindCSS

**Server/Backend:** Node, Express

**Authentication:** JWT & bcrypt

**Database:** MongoDB

**Deployment:** Vercel






## Installation

- Clone the repo
git clone https://github.com/Adithyaaa-07/moviebuzz.git

-  Install dependencies
cd moviebuzz
npm install
cd client
npm install

-  Run backend
npm start

-  Run frontend
cd client
npm start
