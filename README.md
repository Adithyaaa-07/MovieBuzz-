# MovieBuzz-
🎟️ MOVIEBUZZ - Ticket Booking Website (MERN Stack)  A full-stack Ticket Booking Website built using the MERN stack (MongoDB, Express.js, React, Node.js). This project allows users to browse events, book tickets, manage reservations, and handle secure authentication.
🚀 Features
🔑 User Authentication & Authorization (JWT / bcrypt)
🎫 Event Listings & Ticket Booking
📅 Booking History & Management
💳 Payment Integration (Stripe) – in progress
👨‍💻 Admin Panel for event & booking management
📱 Responsive UI with React + Tailwind CSS

🛠️ Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT & bcrypt
Deployment: Vercel
